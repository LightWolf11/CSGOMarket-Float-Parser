# CS:GO Market Bot 🤖

Telegram бот для отслеживания скинов CS:GO на маркетплейсе с фильтрацией по значению float.

## 📋 Возможности

- ✅ Добавление отслеживания предметов по ссылке
- ✅ Фильтрация по диапазону float значений (0.0 - 1.0)
- ✅ Просмотр активных отслеживаний
- ✅ Удаление отслеживания
- ✅ Проверка предметов в реальном времени
- ✅ Уведомления при обновлении цен

## 🚀 Быстрый старт

### Требования

- Python 3.14+
- pip или другой менеджер пакетов Python

### Установка

1. **Клонируйте репозиторий** (или разархивируйте архив):
```bash
cd csgomarket
```

2. **Создайте виртуальное окружение** (если его еще нет):
```bash
python -m venv .venv
```

3. **Активируйте окружение**:

   На Windows:
   ```bash
   .\.venv\Scripts\activate
   ```

   На macOS/Linux:
   ```bash
   source .venv/bin/activate
   ```

4. **Установите зависимости**:
```bash
pip install -r requirements.txt
```

### Конфигурация

1. **Отредактируйте `config.py`**:
   - Укажите ваш `TELEGRAM_BOT_TOKEN` (получите у [@BotFather](https://t.me/botfather))
   - При необходимости измените другие параметры

```python
TELEGRAM_BOT_TOKEN = "ваш_токен_здесь"
CHECK_INTERVAL_MINUTES = 1  # Интервал проверки
MAX_ITEMS_PER_USER = 20      # Максимум предметов на пользователя
```

### Запуск

```bash
python bot_commands.py
```

Вы должны увидеть сообщение:
```
✅ Бот запущен!
📝 Используйте команду /start для начала работы
```

## 💬 Команды бота

| Команда | Описание |
|---------|---------|
| `/start` | Показать приветствие и справку |
| `/track <ссылка> <мин> <макс>` | Добавить отслеживание предмета |
| `/list` | Показать все активные отслеживания |
| `/delete <номер>` | Удалить отслеживание по номеру |
| `/check` | Проверить все отслеживаемые предметы |
| `/clear` | Удалить все отслеживания |

### Примеры использования

**Добавить отслеживание:**
```
/track https://market.csgo.com/ru/Rifle/M4A1-S/M4A1-S%20%7C%20Black%20Lotus%20(Field-Tested) 0.15 0.18
```

**Список отслеживаний:**
```
/list
```

**Удалить отслеживание №2:**
```
/delete 2
```

## ⚙️ Дополнительная конфигурация

### Настройка интервала проверки

В `config.py` измените значение `CHECK_INTERVAL_MINUTES`:
```python
CHECK_INTERVAL_MINUTES = 5  # Проверка каждые 5 минут
```

### Настройка максимальной цены

Установите максимальную цену для уведомлений:
```python
MAX_PRICE_RUB = 50000  # Уведомлять только о предметах до 50000 руб
```

## 🔒 Безопасность

⚠️ **Важно:**
- Никогда не коммитьте файл `config.py` с реальным токеном в публичный репозиторий
- Используйте `.gitignore` для защиты конфиденциальных данных

Пример `.gitignore`:
```
.venv/
__pycache__/
*.pyc
config.py
tracked_items.json
```

## 📊 Формат float значений

- **FN (Factory New)**: 0.00 - 0.07
- **MW (Minimal Wear)**: 0.07 - 0.14
- **FT (Field-Tested)**: 0.14 - 0.37
- **WW (Well-Worn)**: 0.37 - 0.45
- **BS (Battle-Scarred)**: 0.45 - 1.00

## 🐛 Могущие возникнуть ошибки

### "ModuleNotFoundError: No module named 'telegram'"
**Решение:** Убедитесь, что установлены зависимости:
```bash
pip install -r requirements.txt
```

### "Invalid token provided"
**Решение:** Проверьте правильность токена в `config.py`. Получите новый токен у [@BotFather](https://t.me/botfather)

### "Connection refused"
**Решение:** Проверьте подключение к интернету и корректность работы Telegram API

## 📝 Логирование

Бот логирует все события в консоль с форматом:
```
2026-03-26 12:34:56 - telegram.ext._application - INFO - Bot started
```

Для более детального логирования измените уровень в `bot_commands.py`:
```python
logging.basicConfig(
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    level=logging.DEBUG  # Измените с INFO на DEBUG
)
```
