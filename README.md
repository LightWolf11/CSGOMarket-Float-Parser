# CSGO Market Price Tracker Bot

Telegram бот для отслеживания цен на предметы CSGO маркета с поддержкой фильтров по float и стоимости.

## Описание

**CSGO Market Price Tracker** — это Telegram бот, который позволяет отслеживать выбранные предметы на CSGO маркете и получать уведомления при появлении выгодных предложений.

Основные возможности:
- 📍 Добавление и управление отслеживаемыми предметами
- 🎯 Фильтрация по float и цене
- 💬 Индивидуальные уведомления в Telegram
- ⚙️ Гибкая система батчинга уведомлений
- 💾 Сохранение истории найденных предметов и просмотренных лотов
- 🌐 Поддержка разных валют маркета

## Команды бота

- **/start** — начало работы, добавление в отслеживаемые чаты
- **/add** — добавить предмет для отслеживания
  - Пример: `/add AWP Dragon Lore Factory New 0.01 100000 200000`
  - Формат: `/add <название> <минимальный float> <минимальная цена> <максимальная цена>`
- **/list** — список отслеживаемых предметов
- **/pause** — приостановить отслеживание
- **/resume** — возобновить отслеживание
- **/delete** — удалить отслеживаемый предмет
- **/history** — история найденных предметов

## Переменные окружения

| Переменная | Описание | По умолчанию |
|-----------|---------|-------------|
| `TELEGRAM_BOT_TOKEN` | Токен Telegram бота | **обязательно** |
| `CURRENCY` | Валюта маркета (rub, usd, cny и т.д.) | rub |
| `SCAN_INTERVAL_SEC` | Интервал сканирования маркета (сек) | 3 |
| `FILES_PER_SCAN` | Количество файлов маркета для сканирования | 30 |
| `DOWNLOAD_CONCURRENCY` | Количество параллельных загрузок | 8 |
| `INDEX_CACHE_SEC` | Время кеширования индекса маркета (сек) | 120 |
| `MAX_RESULTS_PER_SCAN` | Максимум результатов за одно сканирование | 500 |
| `BATCH_SIZE` | Размер батча уведомлений | 10 |
| `BATCH_MIN_SEND` | Минимум предметов для отправки батча | 3 |
| `BATCH_MAX_WAIT_SEC` | Максимальное время ожидания батча перед отправкой (сек) | 40 |

## Как это работает

1. **Инициализация**: Бот загружается с Telegram Bot API
2. **Обработка команд**: Пользователи добавляют предметы для отслеживания
3. **Сканирование**: Бот периодически сканирует CSGO маркет с интервалом `SCAN_INTERVAL_SEC`
4. **Фильтрация**: Ищет предметы, соответствующие условиям (float, цена, название)
5. **Батчинг**: Собирает найденные предметы в батчи
6. **Уведомления**: Отправляет сообщения в Telegram, когда батч готов или истек timeout
7. **Сохранение**: Сохраняет историю в `data.json`

## Известные особенности

- История найденных предметов ограничивается 500 последними
- История отправленных ограничивается 50,000 последними
- Батчинг оптимизирует отправку уведомлений для исключения спама
- Поддерживается только один тип валюты одновременно (выбирается в .env)
